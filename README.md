# Лесория — статический multipage сайт (Tailwind v3 CLI + Vite dev)

Минималистичный (Apple/Notion) многостраничный сайт для кофейни:
- HTML5 семантика
- Tailwind CSS **без CDN** (Tailwind v3 + CLI)
- Vite используется как удобный dev-server (доступ с телефона по Wi-Fi)
- JS: бургер-меню, фильтр меню, форма, “наверх”, активный пункт меню
- Относительные пути к CSS/JS: `./assets/css/styles.css` и `./assets/js/main.js`
